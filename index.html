Static Website Hosting on AWS

Amazon S3 Public Website

This is a simple static website hosted using Amazon S3. The bucket is configured for static website hosting and public access, allowing users to access content directly via the S3 website endpoint.

This project demonstrates the fundamentals of cloud-based static hosting, including bucket configuration, public access policies, and content delivery using AWS best practices.
